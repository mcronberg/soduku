About Sudoku

🧩 What is this?

A free, client-side Sudoku game that runs entirely in the browser — no server, no account, no install required.

🎯 Why was it built?

This project serves three purposes:

  • A free Sudoku game for everyone
  • A clonable repo for adding your own features
  • A practical example of AI-assisted development

🛠 How is it made?

Pure HTML, CSS and JavaScript — no frameworks, no dependencies. All puzzle generation happens locally in your browser using a backtracking algorithm.

Created by Michell Cronberg with Claude (AI).

How to Play Sudoku

🎯 Objective

Fill every cell in the grid so that each row, column, and box contains all digits exactly once.

  • 9×9 (Standard) — use digits 1–9, boxes are 3×3
  • 6×6 (Mini) — use digits 1–6, boxes are 3×2
  • 4×4 (Kids) — use digits 1–4, boxes are 2×2

Every puzzle has exactly one correct solution.

📜 Rules

  • Each row must contain all digits with no repetition
  • Each column must contain all digits with no repetition
  • Each box must contain all digits with no repetition

🎮 Controls

  • Click/tap a cell to select it
  • Type a number (1-9) or use the number pad
  • Drag & drop a number from the pad onto a cell (works on touch devices too)
  • Press Delete/Backspace or click ✕ to clear a cell
  • Arrow keys to navigate between cells

🎨 Visual Aids

When you select a cell, related cells are highlighted:

  • Selected cell — Strong blue background
  • Same row / column / box — Light blue highlight

A small dot under a number marks your last move.
